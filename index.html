Ayudaremos tanto a profesores como a estudiantes

Checklist es una aplicación web desarrollada para facilitar el registro y control de asistencias en un entorno académico. El sistema está diseñado para ser utilizado tanto por profesores como por estudiantes, permitiendo llevar un seguimiento preciso, organizado y visualmente amigable de la asistencia a clases.

La plataforma permite a los usuarios:

- Registrar asistencias de manera automática y manual.

- Validar la inscripción del estudiante en una asignatura antes de permitir el registro.

- Visualizar los registros filtrados por estudiante y por clase.

- Consultar estadísticas básicas sobre la asistencia a diferentes materias.

Tecnologías utilizadas:

- Frontend: HTML, CSS, JavaScript (Vanilla JS)

- Backend: Node.js con Express

- Base de datos: MySQL
